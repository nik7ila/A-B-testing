# A-B-testing
Email Campaign A/B Testing & Customer Segmentation

# Overview

Simulated an email marketing campaign with A/B testing and customer segmentation to analyze performance across different user groups.

# What I Did
Created synthetic dataset using Python
Segmented users (High / Medium / Low value)
Simulated A/B campaigns
Analyzed open rate, CTR, and conversion
Identified best campaign per segment
# Key Insights
Campaign B performed best for high-value users
Campaign A performed better for low-value users
Engagement strongly influenced open rates
# Tools Used
Python (pandas, numpy)
Excel (pivot tables, charts)
